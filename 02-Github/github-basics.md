- What is GitHub?
		GitHub is an online platform used to store and manage Git repositories.
        It allows developers to upload their local Git projects to the cloud.
        GitHub helps in sharing code, collaborating with others, and maintaining project history.
- Difference between Git and GitHub? 
		Git is a version control system that works locally on a computer.
        GitHub is a remote platform that hosts Git repositories online.
        Git tracks changes, while GitHub helps teams collaborate using those changes.
- Why companies use GitHub?
		Companies use GitHub to manage large projects, track issues, review code,
        and collaborate with developers across the world.
        It improves teamwork, version control, and project safety.
