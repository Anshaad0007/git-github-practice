What is git?
	Git is a version control system used to track changes in files and  projects.
    It helps developers save different versions of their work and go back to previous versions if needed.
    Git is mainly used in software development but can also be used for documentation and notes.
What does git init do?
	The command git init is used to initialize a new Git repository.
    When we run git init inside a folder, Git starts tracking that folder.
What is a repository?
	A repository is a storage space for a project. It contains all project files and their revision history.
